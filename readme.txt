Image Details App

A web-based application that allows users to upload images and view their details, including EXIF data. The application also provides a feature to download the image details in various formats, such as text, CSV, and JSON.

Features:

* Upload images and view their details
* View EXIF data for uploaded images
* Download image details in various formats (text, CSV, JSON)
* Theme switcher to switch between light and dark themes

Technologies Used:

* HTML5
* CSS3
* JavaScript (ES6+)
* EXIF.js library for parsing EXIF data
* Express.js for serving static files

How it Works:

1. Users upload an image file to the application.
2. The application parses the image file and extracts its details, including EXIF data.
3. The application displays the image details and EXIF data to the user.
4. Users can download the image details in various formats by clicking the "Download" button.
5. The application provides a theme switcher to switch between light and dark themes.
